# patent-innovation: 专利创新点发掘

轻量版：搜索 + AI 分析，给出创新方向建议。

## 输入

用户给出一个技术领域关键词，例如：
- "新能源汽车 电池 热管理"
- "AI 大模型 推理 加速"
- "医疗器械 微创 机器人"

## 执行流程

### 1. 搜索专利（WebSearch）

使用 WebSearch 工具（内置搜索），用以下关键词组合搜索：
- `{领域} 专利 site:patents.google.com`
- `{领域} 最新技术 专利`
- `{领域} 创新 技术突破`

> **注意**：使用 WebSearch 工具，不要用 mcp__web-search-local（本地 SearXNG 容器网络受限）

获取前 10 条结果，提取专利名称、申请人、摘要。

### 2. 搜索论文（补充）

搜索最新论文：
- `{领域} 2024 2025 paper arxiv`
- `{领域} state of the art`

### 3. AI 分析

基于收集到的信息，分析：

1. **现有技术格局**：这个领域主要玩家是谁？核心技术路线有哪些？
2. **技术成熟度**：处于什么阶段？早期/中期/成熟期？
3. **创新空白**：有哪些问题还没解决？哪些需求还没被满足？
4. **可操作方向**：具体可尝试的创新点（3-5个）
5. **风险提示**：哪些方向已有专利布局？

### 4. 输出

输出结构化分析报告（Markdown 格式），包含：
- 技术格局概览
- 核心玩家与路线
- 创新机会点（带优先级）
- 行动建议

## 格式约束

- 标题层级：##, ###
- 列表用 `-` 或 `1.`
- 如需表格，用 Markdown 表格格式

## 注意事项

- 专利搜索依赖 Web 搜索，结果可能不完整
- 仅为参考，不能替代专业专利分析
- 重点在于"方向启发"，而非"精确答案"
